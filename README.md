# House Price Prediction

A Machine Learning project to predict house prices using Python and Scikit-learn.

## Project Overview
This project uses the Kaggle House Prices dataset to build a model
that predicts sale prices of houses based on 80 features.

## Results
- **Algorithm:** Random Forest Regressor
- **R² Score:** 0.8925 (89.25% accurate)
- **MAE:** $17,072 average error

## Key Finding
Overall Quality of a house is the most important factor —
it alone accounts for 55.97% of the price prediction.

## Steps Done
1. Data Loading and Exploration (EDA)
2. Data Cleaning — handled missing values, outliers
3. Feature Engineering — Label Encoding, Scaling
4. Model Training — Linear Regression and Random Forest
5. Model Evaluation — MAE, RMSE, R² Score
6. Model Saved using Pickle

## Tech Stack
- Python 3.12
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Jupyter Notebook
